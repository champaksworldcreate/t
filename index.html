2Ô∏è‚É£ What is Pickle?

  • Pickle is a built-in Python module
  • It converts Python objects into binary data
  • Binary data is stored in files
  • The same object can be restored later

One-line formula:

Python Object ‚Üí Bytes ‚Üí File

  3. User inserts or views students
  4. At exit ‚Üí data is saved
  5. Next run ‚Üí data is restored

4Ô∏è‚É£ Code File: pickleutilities.py

This file handles saving and reading data.


import pickle

def saveFile(filename, data):
    with open(filename, 'wb') as file:
        pickle.dump(data, file)

def readFile(filename):
    with open(filename, "rb") as f:
        return pickle.load(f)

Important: dump() saves data, load() restores data.

5Ô∏è‚É£ Code File: test.py

Menu-driven student database program.


import pickleutilities as pt

data = {}
filename = "students.db"

try:
    data = pt.readFile(filename)
except:
    print("No file found. Will create now")

while True:
    option = int(input("0-Exit, 1-Insert, 2-Show\n"))

    if option == 0:
        break

    if option == 1:
        rollno = int(input("Enter Roll No\n"))
        name = input("Enter Name\n")
        data[rollno] = name

    if option == 2:
        for k, v in data.items():
            print(k, v, sep=", ")

pt.saveFile(filename, data)

Most important line:
pt.saveFile(filename, data)
This makes the data permanent.

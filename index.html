1) What is id(variable) in Python?

Important: id() gives the identity of an object (during its lifetime). A variable is only a name that points to that object. 
x = 10
print(id(x))   # identity of the int object 10

Variables are labels (not boxes)

a = 5
b = a
print(id(a), id(b))     # often same: both names point to same object
Quick question üí¨
If a and b have the same id, how many variable names can point to one object?

2) Value of a variable whose id is known

Short answer: You can‚Äôt reliably get ‚Äúthe variable value‚Äù from an id alone, because Python doesn‚Äôt provide a safe reverse mapping of id ‚Üí variable name/value.

Why? One object can have multiple names:

a = 100
b = 100
print(id(a) == id(b))

If you truly need lookup later, design for it:

registry = {}
obj = {"topic": "Python", "time": "7 PM"}
registry[id(obj)] = obj

print(registry[id(obj)])

3) Freeing memory: del and removing references

Rule: Memory is freed when an object has no references left (and cyclic garbage is handled by GC).

‚úÖ Delete a name

x = [1, 2, 3]
del x   # removes the name 'x' (reference)

‚úÖ Reassign to drop the reference

big = [i for i in range(1_000_000)]
big = None   # old list now eligible for cleanup (if no other refs exist)

‚ö†Ô∏è Memory won‚Äôt free if another reference exists

a = [1, 2, 3]
b = a
del a
print(b)  # object still alive because 'b' refers to it

4) Garbage Collector: gc

Python uses:

  • Reference counting for most objects
  • Garbage collection for cycles

Cyclic reference example

a = []
a.append(a)   # a references itself: cycle

Manual trigger (rarely needed):

import gc
gc.collect()

5) Memory used by a variable: shallow vs deep size

Shallow size:

import sys
lst = [1, 2, 3]
print(sys.getsizeof(lst))

Deep size:

import sys

def deep_getsizeof(obj, seen=None):
    if seen is None:
        seen = set()
    obj_id = id(obj)
    if obj_id in seen:
        return 0
    seen.add(obj_id)

    size = sys.getsizeof(obj)

    if isinstance(obj, dict):
        size += sum(deep_getsizeof(k, seen) + deep_getsizeof(v, seen)
                    for k, v in obj.items())
    elif isinstance(obj, (list, tuple, set)):
        size += sum(deep_getsizeof(i, seen) for i in obj)

    return size

data = [1, 2, [3, 4], {"a": 10, "b": [11, 12]}]
print(deep_getsizeof(data), "bytes")

6) Time needed for execution

Precise timing with perf_counter

import time

start = time.perf_counter()
x = sum(range(1_000_000))
end = time.perf_counter()

print("Time:", end - start, "seconds")

Accurate benchmarking with timeit

import timeit
t = timeit.timeit("sum(range(1000))", number=10000)
print("Total time:", t)

Quick profiling

import cProfile
cProfile.run("sum(range(1_000_000))")

Use for when you want to repeat something a fixed number of times or go through items one by one.

Concept

1) What is a for loop?

A for loop takes items one by one from a sequence (like range, list, or string) and runs the block for each item.

for item in sequence:
    # repeated code
Think: "For each item, do this".
range()

2) range() basics

Print 1 to 5

for i in range(1, 6):
    print(i)

Stop value 6 is not included, so it prints 1..5.

Even numbers with step

for i in range(2, 11, 2):
    print(i)

Start 2, step 2, stops before 11.

Example

4) Loop through a string

name = "Python"
for ch in name:
    print(ch)

Each character is taken one by one from the string.

Control

5) break and continue

# break example
for i in range(1, 10):
    if i == 5:
        break
    print(i)continue example

for i in range(1, 6): if i == 3: continue print(i)
